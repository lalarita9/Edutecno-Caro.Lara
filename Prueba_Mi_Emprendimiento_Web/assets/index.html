HTML y CSS para principiantes.

HTML y CSS son dos herramientas importantes para dar inicio en el mundo del desarrollo web. Por lo tanto, en este cuso aprenderás todo sobre HTML y CSS, desde lo más básico hasta un nivel intermedio, logrando desarrollar tu primera página web responsiba, sin la necesidad de aplicar liberías externas para CSS.

Estilos con SASS.

SASS es una herramienta importante hoy en día para todo desarrollador web, ya que permite crear estilos más ordenados, eficientes y con menos líneas de código. Por ello, en este curso aprenderás todo lo necesario para crear hojas de estilos desde la sintáxis de SCSS, implementando técnicas propias de SASS.
